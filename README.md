GIT BASH - 
cd = entrar
ls = listar
git init = chama o git
git branch -M main = tornar-se o main
git add . = preparar para receber alguma alteração
git commit = criar uma mensagem
git status = mostrar informações importantes 
git push -u origin = leva as alterações do repositório
code . = abrir o VScode

VSCODE -
Exibir comando na tela = System.out.println
int = númeors inteiros sem casa decimal / ex: 10, -5, 1000
double = números com casas decimais / ex: 10.5, 3.14, -7.85
boolean = valores lógicos verdadeiro ou falso / ex: true ou false
char = um único caractere / ex: "z", "a", "@"
String = sequência de caracteres / ex: "olá, gabriel!" (String = S maiúsculo)
